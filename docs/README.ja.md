## /write

```
         # はじめに

         これは**シンプル**なMarkdownエディターです
         リアルタイムプレビューとエクスポート対応

         > ヒント: 書式設定にMarkdown構文を使用

         `const greeting = "こんにちは、世界！";`

         - 軽量で高速
         - トラッキングや広告なし
         - データはローカルに保存
```

## /room

```
         # リアルタイムコラボレーション

         これは Yjs と WebRTC による
         **協調エディター**です

         > ヒント: 同じURLを複数のタブで開いて
         > コラボレーションしましょう

         - WebRTC P2Pによるリアルタイム同期
         - emojiインジケーター付きリモートカーソル追跡
         - CRDTによる自動競合解決
         - サインアップ不要、即座にコラボレーション可能
```

## /jump

```
         # ピクセルジャンプアドベンチャー

         これはHTML5 Canvasで構築された
         **レトロスタイル的平台跳躍ゲーム**です

         > ヒント：コインを収集し、敵を避けよう！

         - 手続き的に生成されるプラットフォーム
         - 進行に伴う難易度の向上
         - レトロなピクセルアート graphics
         - モバイル向けのタッチコントロール対応
```


## /story

```
         # テキストアドベンチャー

         これは**インタラクティブなテキストアドベンチャーゲーム**で、
         分岐するストーリーラインと選択肢があります

         > ヒント：慎重に選択してください - それらは重要です！

         - 複数のエンディングを持つ分岐ナラティブ
         - 18言語サポート
         - 美しいグラデーションデザイン
         - モバイルフレンドリーなレスポンシブレイアウト
```
