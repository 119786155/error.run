# error.run
osobisty projekt do nauki, ćwiczeń i eksploracji

## /

```
         Pył

         Wiatr przechodzi
         Bez śladu
         Serce zachowuje
         Jeden oddech
```

## /write

```
         # Pierwsze kroki

         To jest **minimalny** edytor markdown
         z podglądem w czasie rzeczywistym i Eksportem

         > Podpowiedź: Użyj składni markdown do formatowania

         `const greeting = "Witaj świecie!";`

         - Lekki i szybki
         - Brak śledzenia lub reklam
         - Twoje dane pozostają lokalnie
```

## /room

```
         # Współpraca w czasie rzeczywistym

         To jest **edytor współpracy** zasilany
         przez Yjs i WebRTC

         > Podpowiedź: Otwórz ten sam adres URL w wielu kartach, aby współpracować

         - Synchronizacja w czasie rzeczywistym przez WebRTC P2P
         - Śledzenie zdalnego kursora z wskaźnikami emoji
         - Automatyczne rozwiązywanie konfliktów za pomocą CRDT
         - Bez rejestracji, natychmiastowa współpraca
```

## /jump

```
         # Pixelowa Przygoda ze Skakaniem

         To jest **gra platformowa w stylu retro**
         zbudowana z HTML5 Canvas

         > Podpowiedź: Zbieraj monety i unikaj wrogów!

         - Proceduralnie generowane platformy
         - Stopniowe zwiększanie trudności w miarę postępów
         - Grafika w stylu pixel-art
         - Sterowanie dotykowe dla obsługi mobilnej
```

## /story

```
         # Przygoda Tekstowa

         To jest **interaktywna gra przygodowa tekstowa**
         z rozgałęzionymi liniami fabuły i wyborami

         > Porada: Podejmuj wybory starannie - one mają znaczenie!

         - Rozgałęzioną historię z wieloma zakończeniami
         - Obsługa 18 języków
         - Piękny projekt z gradientem
         - Responsywny układ przyjazny dla telefonów
```
