# error.run
โปรเจกต์ส่วนตัวสำหรับการเรียนรู้ ฝึกฝน และสำรวจ

## /

```
         ฝุ่น

         ลมพัดผ่าน
         ไม่มีร่องรอย
         หัวใจเก็บรักษา
         หนึ่งลมหายใจ
```

## /write

```
         # เริ่มต้นใช้งาน

         นี่คือโปรแกรมแก้ไข markdown **ที่เรียบง่าย**
         พร้อมการแสดงตัวอย่างแบบเรียลไทม์ และการส่งออก

         > เคล็ดลับ: ใช้ไวยากรณ์ markdown สำหรับการจัดรูปแบบ

         `const greeting = "สวัสดีโลก!";`

         - เบาและรวดเร็ว
         - ไม่มีการติดตามหรือโฆษณา
         - ข้อมูลของคุณยังคงอยู่ในเครื่อง
```

## /room

```
         # การทำงานร่วมกันแบบเรียลไทม์

         นี่คือโปรแกรมแก้ไขที่**ช่วยให้ทำงานร่วมกัน**
         ขับเคลื่อนด้วย Yjs และ WebRTC

         > เคล็ดลับ: เปิด URL เดียวกันในแท็บหลายเพื่อทำงานร่วมกัน

         - การซิงค์แบบเรียลไทม์ผ่าน WebRTC P2P
         - การติดตามเคอร์เซอร์ระยะไกลด้วยตัวบ่งชี้ emoji
         - การแก้ไขขัดแย้งอัตโนมัติด้วย CRDT
         - ไม่ต้องลงทะเบียน ทำงานร่วมกันทันที
```

## /jump

```
         # ผจญภัยกระโดดพิกเซล

         นี่คือเกมแพลตฟอร์ม **สไตล์เรโทร**
         ที่สร้างด้วย HTML5 Canvas

         > เคล็ดลับ: เก็บเหรียญและหลีกเลี่ยงศัตรู!

         - แพลตฟอร์มที่สร้างขึ้นด้วยขั้นตอน
         - ความยากที่เพิ่มขึ้นตามความก้าวหน้า
         - กราฟิกด้วยศิลปะพิกเซลสไตล์เรโทร
         - ตัวควบคุมด้วยสัมผัสสำหรับการสนับสนุนมือถือ
```
