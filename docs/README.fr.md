## Room

```
         # Collaboration en Temps Réel

         Ceci est un **éditeur collaboratif**
         propulsé par Yjs et WebRTC

         > Conseil : Ouvrez la même URL dans plusieurs
         > onglets pour collaborer

         - Synchronisation en temps réel via WebRTC P2P
         - Suivi du curseur distant avec indicateurs emoji
         - Résolution automatique des conflits avec CRDTs
         - Pas d'inscription requise, collaboration instantanée
```
