## /room

```
         # Реальна-часова співпраця

         Це **редактор співпраці**, що працює
         за допомогою Yjs та WebRTC

         > Підказка: Відкрийте той самий URL у кількох вкладках, щоб співпрацювати

         - Реальна-часова синхронізація через WebRTC P2P
         - Відстеження віддаленого курсора з індикаторами emoji
         - Автоматичне вирішення конфліктів за допомогою CRDT
         - Без реєстрації, миттєва співпраця
```
