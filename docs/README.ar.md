# error.run
مشروع شخصي للتعلم والممارسة والاستكشاف

## /

```
         الغبار

         الريح تمر
         لا أثر
         القلب يحتفظ
         نَفَس واحد
```

## /write

```
         # البدء

         هذا محرر **بسيط** Markdown
         مع معاينة مباشرة والتصدير

         > نصيحة: استخدم صيغة Markdown

         `const greeting = "مرحبا، العالم!";`

         - خفيف وسريع
         - بدون تتبع أو إعلانات
         - بياناتك تبقى محلياً
```

## /room

```
         # التعاون في الوقت الفعلي

         هذا **محرر تعاوني** يعمل بتقنية Yjs و WebRTC

         > نصيحة: افتح نفس URL في عدة علامات تبويب للتعاون

         - مزامنة فورية عبر WebRTC P2P
         - تتبع المؤشر عن بعد مع مؤشرات emoji
         - حل تلقائي للنزاعات باستخدام CRDTs
         - لا حاجة للتسجيل، تعاون فوري
```

## /jump

```
         # مغامرة قفز البكسل

         هذه **لعبة منصات بأسلوب رجعي**
         مبنية باستخدام HTML5 Canvas

         > نصيحة: اجمع العملات وتجنب الأعداء!

         - منصات مُولَّدة إجرائياً
         - صعوبة متصاعدة مع التقدم
         - رسومات بكسل رجعية
         - تحكم باللمس لدعم الأجهزة المحمولة
```
