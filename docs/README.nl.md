## Write

```
         # Aan de slag

         Dit is een **minimale** markdown-editor
         met real-time voorbeeld en Exporteren

         > Tip: Gebruik markdown-syntaxis voor opmaak

         `const greeting = "Hallo wereld!";`

         - Lichtgewicht en snel
         - Geen tracking of advertenties
         - Uw gegevens blijven lokaal
```

## Room

```
         # Real-time Samenwerking

         Dit is een **samenwerkingseditor** aangedreven
         door Yjs en WebRTC

         > Tip: Open dezelfde URL in meerdere tabbladen om samen te werken

         - Real-time synchronisatie via WebRTC P2P
         - Externe cursor-tracking met emoji-indicatoren
         - Automatische conflictoplossing met CRDT's
         - Geen aanmelding vereist, onmiddellijke samenwerking
```
