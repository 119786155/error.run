## /write

```
         # 시작하기

         이것은 실시간 미리보기와
         내보내기 기능이 있는 **간단한** Markdown
         편집기입니다

         > 팁: 서식 지정에 Markdown 구문 사용

         `const greeting = "안녕하세요, 세계!";`

         - 가볍고 빠름
         - 추적이나 광고 없음
         - 데이터는 로컬에 저장됩니다
```

## /room

```
         # 실시간 협업

         이것은 Yjs와 WebRTC를 기반한
         **협업 편집기**입니다

         > 팁: 여러 탭에서 같은 URL을 열어 협업하세요

         - WebRTC P2P를 통한 실시간 동기화
         - emoji 표시기가 있는 원격 커서 추적
         - CRDT를 통한 자동 충돌 해결
         - 회원가입 불필요, 즉시 협업 가능
```

## /jump

```
         # 픽셀 점프 어드벤처

         이것은 HTML5 Canvas로 구축된
         **레트로 스타일 플래т포머 게임**입니다

         > 팁: 코인을 모으고 적을 피하세요!

         - 절차적으로 생성되는 플랫폼
         - 진행함에 따라难度 증가
         - 레트로 픽셀 아트 그래픽
         - 모바일 지원 터치 컨트롤
```
